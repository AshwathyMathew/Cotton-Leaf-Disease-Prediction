# Cotton-Leaf-Disease-Prediction
To classify the cotton leaf disease
Steps For Creating Image Processing Project			
1. Download the dataset (from Kaggle)	https://www.kaggle.com/datasets/seroshkarim/cotton-leaf-disease-dataset/data	Dataset info	
2. Extract the dataset	In our dataset, there are 4 folders. One is a healthy leaves folder and the other three are diseased leaves folder	Bacterial Blight	448
3.Type the coding 	Use tensor flow and open CV	
4. Train the model	Trained model is saved for testing purposes
5. Accuracy and Loss Chart is Displayed
6. Prediction	Give one input image and get the result
			
Create the flask application for user friendly
